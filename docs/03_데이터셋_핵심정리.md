# 데이터셋 핵심 정리 (공식 문서 분석)

> 출처: AI-Hub 공식 `데이터설명서.hwp`, `23년 활용 가이드라인 v1.0`(2023.12.29, ㈜피씨엔)
> 추출 원문: `docs/참고_데이터설명서.txt`, `docs/참고_활용가이드라인.txt`

## 1. 공식 베이스라인 성능 (★ 우리 보고서의 비교 기준)
| TASK | 모델 | 지표 | 목표치 | 공식 결과 |
|------|------|------|--------|-----------|
| 이미지 분류(균열 종류) | **CvT** (Wu+2022) | F1-score | 80% 이상 | **84.67%** |
| 이미지 분할(균열 탐지) | **SegFormer** (Xie+2021) | mIoU | 55% 이상 | **55.78%** |

→ 우리 프로젝트의 성능 목표를 "공식 베이스라인(F1 84.67% / mIoU 55.78%) 대비"로 잡으면 보고서 설득력이 높아진다.

## 2. 공식 모델 vs 우리 선택
- **공식**: 분류(CvT) + 분할(SegFormer) **두 모델 분리**.
- **우리**: **YOLOv11-seg 단일 모델** (탐지 + 인스턴스 분할 동시).
  - 장점: 실시간, 모바일 친화, 하나로 위치+종류+면적 동시 산출 → 현장 앱에 적합.
  - 보고서 포지셔닝: "공식 2-모델 파이프라인을 단일 실시간 모델로 통합, 현장 적용성 강화" + 공식 수치와 비교.

## 3. 공식 학습 조건 (참고용)
- 환경: Python 3.8, CUDA 12.2, PyTorch 2.1.1, GPU **RTX A6000**, RAM 256GB, Ubuntu 22.04
- CvT: batch=16, optimizer=SGD, max_epoch=500(early stop), lr 1e-4→1e-6, CosineAnnealingWarmRestarts
- 데이터 분할: **학습 80% / 검증 10% / 평가 10%** (419,995 / 52,500 / 52,505)
- → 우리는 Colab(T4)로 소형부터. 분할 비율은 공식과 동일하게 80/10/10 권장.

## 4. 어노테이션 포맷 (변환 스크립트와 일치 확인됨)
`image[].annotations[]` 배열, 각 객체:
| 필드 | 설명 |
|------|------|
| `label` | 균열 종류 (crack, reticular crack, detachment, spalling, efflorescence, leak, rebar, material separation, exhilaration, damage) |
| `labelNum` | 라벨 번호 |
| `points` | 좌표 `[[x1,y1],[x2,y2],...]` |
| `width` / `px` | **균열 폭 / 폭 픽셀값** ← 위험도 산정 핵심 입력 |
| `shape` | `Polygon`(면적 결함) 또는 `Polyline`(가는 실금) |
- `object_included`(Y/N): 균열 유무. N이면 annotations 없음(균열 미포함 25,000건).
- → `data-tools/aihub_to_yolo.py`가 이 구조를 처리하도록 작성됨. 특히 `width/px`는 향후 위험도 엔진에 직접 연결 가능.

## 5. 클래스 불균형 (재확인)
`균열` 80.38%(401,908건)로 압도적, 나머지 9종은 각 0.2~3.8%.
→ 학습 시 focal loss / class weight / 소수클래스 oversampling 필수.
(주의: 설명서와 웹페이지의 일부 클래스 수치가 상이 — 망상균열/들뜸/박락 등. 실제 라벨 분포는 다운로드 후 스크립트로 재집계 권장.)

## 6. 공식 서비스 활용 시나리오 (보고서 '기대효과'에 인용 가능)
- AI 기반 SOC 시설물 성능진단 모니터링 도구의 기초 데이터
- 균열 길이·넓이 등 정도에 따른 유지보수 범위 다각화
- 유지보수 비용·일정 계산을 위한 협의 데이터
- 인력 진입이 어려운 위험지역에 촬영장비 투입 → 균열 진단 AI 및 탐지 로봇
- 정기점검 기반 AI 균열 측정 및 **상태예측** 모델, 유지·보수 **의사결정 지원**
