# 실행 가이드 (제일 먼저 읽기)

전체 시스템을 내 PC에서 켜는 방법. **모델 학습 전에도 mock 모드로 전부 작동**합니다.

## 0. 사전 준비 (한 번만)
- Python 3.12 (이미 설치됨: `py -3.12`)
- Node.js (이미 설치됨)

## 1. AI 서버 켜기
PowerShell 새 창에서:
```powershell
cd "C:\Users\doeun\OneDrive\Desktop\Google_캡스톤디자인\ai-server"
# 최초 1회만:  py -3.12 -m venv .venv ;  .\.venv\Scripts\python.exe -m pip install -r requirements.txt
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --port 8000
```
- 확인: 브라우저에서 http://localhost:8000/docs (API 문서가 보이면 성공)

## 2. 웹 켜기
또 다른 PowerShell 창에서:
```powershell
cd "C:\Users\doeun\OneDrive\Desktop\Google_캡스톤디자인\web"
# 최초 1회만:  npm install
npm run dev
```
- 확인: 브라우저에서 http://localhost:5173
  - **관리자 대시보드**: 지도/통계/보수 우선순위
  - **현장 촬영**: 사진 올리면 AI 분석 결과

## 3. 휴대폰으로 촬영 테스트 (선택)
1. PC와 폰을 **같은 와이파이**에 연결.
2. `web/.env` 파일을 만들고 PC의 IP를 적기:
   ```
   VITE_API_BASE=http://192.168.x.x:8000
   ```
   (IP는 `npm run dev` 실행 시 Network 주소 참고)
3. 폰 브라우저에서 `http://192.168.x.x:5173` 접속 → '현장 촬영'에서 카메라로 촬영.

## 4. 진짜 AI 모델 끼우기 (Colab 학습 후)
1. `notebooks/02_AIHub_데이터_학습.ipynb` 로 Colab에서 AI-Hub 데이터 학습 → `best.pt` 다운로드.
2. 그 파일을 `ai-server/models/best.pt` 에 넣기.
3. ultralytics 설치: `.\.venv\Scripts\python.exe -m pip install ultralytics`
4. AI 서버 재시작 → 자동으로 진짜 추론 모드로 전환 (상단 배지가 'AI 모델 연결됨'으로 바뀜).

## 폴더 구조
```
├── ai-server/      AI 추론 서버 (FastAPI)
│   ├── app/        서버 코드 (탐지·위험도·DB·API)
│   ├── models/     여기에 best.pt 넣기

├── web/            웹 (현장 촬영 + 관리자 대시보드)
├── data-tools/     AI-Hub → YOLO 변환 스크립트
├── notebooks/      Colab 학습 노트북
└── docs/           로드맵·아키텍처 문서
```
